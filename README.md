Dockerのビルド
```
docker-compose build
```

コンテナの起動
```
docker-compose up -d
```

コンテナの停止
```
docker-compose down
```
未使用コンテナの削除
```
docker system prune -a
```

javaのコンテナに入る
```
docker exec -it java /bin/bash
```

gradleプロジェクトの作成
```
gradle init
```

```
./gradlew run
```

クリーン
```
gradle clean
```
